# Чек-лист: Функция добавления воды

## Предусловия:
- Пользователь авторизован
- Открыт раздел "Вода"

## Тест-кейсы:
- [ ] Добавление стандартного объема (250 мл)
- [ ] Ручной ввод объема воды
- [ ] Проверка сброса добавленной воды
- [ ] Изменение единиц измерения (мл → л)
- [ ] Работа без интернета
